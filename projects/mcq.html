Currently Building an MCQ website.

Currently working on an MCQ website through which anyone can train skills in specific subjects.

Why ?

While I was preparing for interviews, I found that most of the resources are paid through which we can prepare interview questions. However there are some free platforms available. So, I decided to build my own small application through which anyone can train skills in different subjects.

I have used following tech stack in this project -

  • MongoDb
  • Angular and ExpressJs
  • Repository Architecture pattern.
  • cyclic.sh for deployment

Main difficulties that I have solved while building this project.

  • While I was building the backend side of this project. Cyclic.sh had changed their policy (You can only host 1 website/github account). So I have solved that problem by creating nested application and grouped that application into a single one so that I can host all of my future projects using single repository.
  • Another thing was that each application should have their own mongodb cluster so I have solved that issue as well.
  • Planned sprints over JIRA so that I can easily tracked issues/features.
